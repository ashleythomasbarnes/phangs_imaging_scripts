############
Distance key
############

The distance key maps from target names to distance in Mpc.

When the user requests to convolve to specific physical resolutions
as part of derived product creation a physical distance is
needed. This file provides that distance.

This file should be comma delimited. Each entry should have **2** columns,
as described below.

- **Column 1:** Target name.
- **Column 2:** Distance to target in Mpc.
